# الفراغ المبدع (Creative Void) - v4.0
## صياغة منطقية ورياضية لنشوء الوجود من استحالة العدم

هذا المشروع هو مختبر فيزيائي معلوماتي رقمي يهدف إلى تحويل نظرية **"الفراغ المبدع"** من مجرد صياغة فلسفية ورياضية إلى محاكاة برمجية ملموسة.

### ملخص النظرية
تطرح نظرية الفراغ المبدع أن الوجود ليس حادثة عشوائية، بل نتيجة لازمة منطقياً لاستحالة العدم المطلق. من خلال ثلاث طبقات مترابطة (منطقية، رياضية، ومعلوماتية)، تثبت النظرية كيف ينشأ التمايز، ومن ثم البنية، والقوانين، وصولاً إلى الكتلة والوعي.

### هيكلية المشروع البرمجي
تم تقسيم المشروع إلى وحدات برمجية تعكس فصول النظرية:
1. **`core_physics.py`**: يطبق معادلات الفصل الثالث (حقل التمايز) وآلية كسر التناظر (Bifurcation).
2. **`information_mass.py`**: يطبق الفصل الرابع (مبدأ لانداور) لحساب المكافئ الكتلي لمعالجة المعلومات.
3. **`spacetime_weaver.py`**: يطبق الفصل الخامس وآلية اضمحلال المعلومات عبر الحدود (BIA) لتفسير نشوء المسافة والزمكان.
4. **`consciousness_metrics.py`**: يطبق الفصل السادس ومعادلة كثافة الوعي ذاتية المرجعية.
5. **`main.py`**: الملف التنفيذي الذي يربط المحاكاة ويولد النتائج البصرية والإحصائية.

### كيفية التشغيل
1. تأكد من تثبيت المكتبات المطلوبة:
   ```bash
   pip install numpy matplotlib scipy
   ```
2. قم بتشغيل المحاكاة الرئيسية:
   ```bash
   python main.py
   ```

### النتائج المتوقعة
عند تشغيل المحاكاة، ستلاحظ:
- **كسر التناظر**: عندما يتحول معامل العتبة الحرجة ($\alpha$) من السالب إلى الموجب، يظهر التمايز في الحقل (ارتفاع التباين).
- **الكتلة المعلوماتية**: يزداد المكافئ الكتلي مع زيادة تعقيد وتمايز الحقل.
- **نشوء الوعي**: تبدأ كثافة الوعي بالارتفاع بمجرد استقرار التمايزات وبدء التكامل المعلوماتي.
